  5. The project repo contains a README
  6. Site is successfully deployed to GitHub pages.
  7. Project is in a polished, portfolio-quality state.
  8. Required content was in place by the Friday deadline.
  9. Project demonstrates an understanding of this week's concepts. If prompted, you can discuss your code with an instructor using the correct terminology.
